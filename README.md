# DUNK Archive 96-081

Мобильная интерактивная веб-игра «Незакрытый счёт. 30 лет».

## Запуск

Откройте `index.html` в браузере или опубликуйте папку как статический сайт.

Для этапа с микрофоном браузеру обычно нужен `localhost` или HTTPS.

## Команды

```bash
npm run build
npm run serve
```

`npm run build` проверяет, что все файлы приложения на месте. Для деплоя не нужны секретные ключи и переменные окружения.

## Состав

- `index.html` — точка входа.
- `styles.css` — мобильный визуальный стиль.
- `app.js` — логика игры, сохранение прогресса и интерактивные этапы.
